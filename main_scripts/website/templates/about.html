About This Capstone Project


Is there a way to estimate the likelihood of traffic accidents using publically available data? If this information is available, it will be possible to optimize emergency response resources by correctly allocating personnels at areas with anticipated risk.

In order to solve this question, I have attempted to utilize Data Science techniques to search for potential features that are predictive of traffic accident, and to plot the likelihood of accidents in Seattle precincts.

All scripts and detailed explanation can be found in [Github link].


Data Sources:

  • Socrata 911 Response Records
  • Wunderground Weather History
  • Wunderground Weather API
  • Seahawks Games Schedule
  • Mariners Games Schedule
